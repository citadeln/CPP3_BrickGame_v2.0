◆ chech_timer()

void chech_timer ( Params_t *  prms)

Функция проверки истечения времени для движения вниз.

Функция проверяет разницу между текущем временем и временем начала движения вниз. Если разница больше текущего значения скорости, то стадия игры переходит в шифтинг (принудительный сдвинг вниз на 1 строку).

Аргументы
prmsУказатель на структуру со всеми данными по игре.

◆ create_field_info()

GameInfo_t* create_field_info ( Figureinfo_t *  figure_info)

Функция создает структуру с начальными данными для поля и статистики.

Функция создает:

  • массив для поля 20+2х10 - 2 дополнительные строки отводятся под появление фигуры и дальнейший спуск вниз на 2 строки, для проверки условия gameover.
  • массив для поля будущей фигуры 4х4 Функция заполняет начальные данные для статистики:
  • счет,
  • рекорд,
  • уровень,
  • скорость(в наносекундах),
  • состояние паузы.

Значения паузы:

  • -1 - конец игры,
  • 0 - игра идет,
  • 1 - пауза в игре,
  • 2 - начало игры,
  • 3 - gameover.
Аргументы
figure_infoПринимает указатель на структуру фигуры для заполнения массива будущей фигуры.
Возвращает
GameInfo_t Возвращает указатель на статичную структуру.

◆ create_prms()

Params_t* create_prms ( Params_t *  pointer)

Функция, создающая основную статичную структуру, хранящую данные о состоянии игры.

При первом запуске создает статичную структуру со всеми данными о текущем состоянии игры, заполняет ее стартовыми значениями, и отдает на нее указатель.

Структура содержит:

  • указатель на структуру с данными о поле и статистике,
  • указатель на структуру с данными о фигурах,
  • текущее состояние игры,
  • время для срабатывания движения вниз.

При последующих запусках отдает только указатель на структуру.

Аргументы
pointerуказатель, если передается NULL, то структура создается со стартовыми значенями. Используется при первом запуске или начале новой игры после gameover.
Возвращает
Params_t* Возвращает указатель на статичную структуру.
